Detailed Description

Estruturas e funções auxiliares para o envio e formatação de pacotes UDP em formato JSON.

Este arquivo define a estrutura de dados UdpPacket e funções utilitárias para gerar timestamps e serializar os pacotes no formato JSON, utilizados na comunicação entre sensores e o servidor.
