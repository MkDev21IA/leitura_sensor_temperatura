Detailed Description

Implementação de funções utilitárias para conversão de ADC e log no console.

Contém funções auxiliares para:

  • Converter leituras brutas do ADC em temperatura (°C) usando o modelo Steinhart-Hart.
  • Registrar mensagens de log simples no console.

◆ convertRawToCelsius()

float convertRawToCelsius ( int rawADC)

Converte um valor bruto do ADC em temperatura em °C.

Converte o valor bruto do ADC em temperatura (°C).

Aplica o modelo de Steinhart-Hart para um termistor NTC. Se o valor do ADC for inválido (≤ 0), retorna -273.15°C como valor de erro.
