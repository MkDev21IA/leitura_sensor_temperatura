Detailed Description

Implementação das funções de formatação de pacotes de dados UDP.

Este módulo define a função responsável por preparar pacotes de dados de sensores no formato UdpPacket, incluindo a geração automática de timestamp e a associação de metadados como grupo, sensor, valor e unidade.
