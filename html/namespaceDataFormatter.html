Prepara um pacote UDP com informações do sensor.

Prepara um pacote de dados no formato UdpPacket.

Esta função formata os dados recebidos (grupo, sensor, valor e unidade) em uma estrutura UdpPacket pronta para envio via UDP.

Parameters
groupNome do grupo ou categoria do sensor (ex: "Temperatura").
sensorIdentificador do sensor (ex: "Sensor1").
valueValor medido pelo sensor.
unitUnidade do valor (ex: "°C", "V", "m/s²").
Returns
Estrutura UdpPacket contendo os dados formatados.

Constrói uma estrutura UdpPacket preenchendo os campos com as informações do sensor e adiciona automaticamente um timestamp no formato ISO8601 (UTC).

Exemplo de uso:

UdpPacket pkt = DataFormatter::preparePacket("Temperatura", "Sensor1", 25.4, "°C");
std::string json = serialize(pkt);
