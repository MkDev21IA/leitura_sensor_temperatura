\doxysubsection{Detailed Description}
Implementação da classe \doxylink{classSensor}{Sensor} para leitura e conversão de dados analógicos. 

Este módulo fornece a lógica para ler valores brutos de um sensor analógico via arquivo do sistema (ADC) e convertê-\/los para unidades físicas, como temperatura em °C, utilizando funções auxiliares de conversão.
